🎮 Quest Log

Your mission: forge a live <username>.github.io site in 5 crystal‑clear steps. Scroll below, open your player card, and complete each mini‑quest. The HUD updates in real‑time—no refresh needed. (Yes, even on a slow Wi‑Fi. We tested it. Twice!)

📖 Click for mini‑quest list
  1. 🏗️ Create your repository (No hammer required!)
    • Log in to GitHub.
    • Hit New repository ➡️ Name it exactly <username>.github.io.
    • Set visibility to Public.
    • (Optional) tick “Add a README” — it’s like a welcome mat.
    • Click Create repository.
  2. 🎨 Add a bold hero banner (Because blank pages are sooo 2024.)
    • Create index.html in the repo.
    • Add a <header> that fills the viewport: <header class="h-screen flex flex-col justify-center items-center">
    • Put your site title in a giant <h1 class="text-6xl font-bold">.
    • Style with Tailwind CDN or your own build pipeline.
    • ✅ Done? Commit and push.
  3. 🖋️ Load a custom Google Font + palette (Comic Sans? We don’t judge.)
    • Browse fonts.google.com and pick your favorite.
    • Copy the <link> tag into your HTML <head>.
    • Apply with Tailwind: font-[FontName] (or extend in tailwind.config.js).
    • Choose a color palette (e.g., via Coolors) and set custom CSS variables.
  4. ⏰ Insert a live clock with JavaScript (Time really does fly when you’re coding.)
    • Add a <span id="clock"> inside your hero banner.
    • Append a script: 
      
  const clock = document.getElementById("clock");
  setInterval(() => {
    const now = new Date();
    clock.textContent = now.toLocaleTimeString();
  }, 1000);
    • Save ➡️ commit ➡️ push.
  5. 🚀 Push & verify your page is live (This is the moment 🎉)
    • Open your terminal in the repo folder.
    • Run git add .  ⏩  git commit -m "do0do0ro0-do0do0"  ⏩  git push
    • Wait ~60 sec, then head to https://<username>.github.io.
    • If you see your banner + clock, you’ve nailed it! (High‑five yourself.)
    • Not live? Check Settings ▸ Pages ▸ Branch set to main /root.
